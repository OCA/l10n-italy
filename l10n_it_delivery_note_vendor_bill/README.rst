**English**

This module allows users to autocomplete an invoice from a Delivery Note

**Italiano**

Questo modulo consente agli utenti di auto-completare una fattura a partire da un DdT.

Configuration
=============

**English**

No configuration required after module installation.

**Italiano**

Dopo l'installazione del modulo non sono necessarie ulteriori configurazioni.

Usage
=====

**English**

In vendor bills, under the field "Autocomplete" there is a new field "Autocomplete from DN" whose behaviour is analogous to the old one.

**Italiano**

Nelle fatture fornitore, sotto il campo "Autocompleta" c'è un nuovo campo "Autocompleta da DdT" il cui comportamento è analogo al precedente.
